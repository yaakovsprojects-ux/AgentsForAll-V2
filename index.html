01 — what is it

An AI that doesn't just chat —
it does things

A regular chatbot answers questions. An AI agent can plan, make decisions, and take action on your behalf — like having a tireless digital assistant.

Agent loop
You→Give a goal: "Book me a dentist"
↓
Think→Break the goal into steps
↓
Act→Search, fill forms, send emails
↓
Check→Did it work? If not, retry
↓
Done→Reports the result back to you
  • 🧠
    It thinks step by stepAn agent breaks a big goal into smaller tasks automatically — you don't micromanage it.
  • 🔧
    It uses toolsAgents can browse the web, run code, read files, send emails, and call APIs — all on your command.
  • 🔁
    It loops until it succeedsIf something doesn't work, it retries with a different approach rather than giving up.
  • 👀
    You stay in controlYou set the goal and the limits. The agent asks before acting on anything sensitive.

03 — how it works

Under the hood,
in plain English

You don't need to know this to use an agent — but understanding it makes you a smarter user.

01

You give it a goal

Instead of typing a question, you describe an outcome: "Summarize all unread emails from this week and sort them by urgency."

02

The AI makes a plan

The language model thinks through the steps: connect to email, filter by date, read each message, categorize, then summarize.

03

It calls tools to take action

Tools are apps the agent can use — a web browser, a code runner, your calendar, a file reader. It picks the right one for each step.

04

It checks its own work

After each step the agent reviews the result. If something went wrong, it backtracks and tries a different approach.

05

It delivers the result to you

Once the goal is achieved, the agent presents what it did and found. You can ask follow-ups or hand it the next task.

04 — install guide

Get an agent running
on your device

Pick your platform. Most people are up and running in under 10 minutes.

windows — Claude Desktop + MCP
  1. Download Claude DesktopGo to claude.ai/download and grab the Windows installer. Double-click the .exe and follow the prompts.
  2. Sign in or create a free accountUse your email or a Google account. No credit card required for the free tier.
  3. Enable agent features in SettingsOpen the app → click your avatar → Settings → Features → turn on Tools & Computer Use.
  4. Try your first agent taskType: Search the web for the best free note-taking apps and give me a ranked list.
  5. (Optional) Add MCP serversMCP lets agents connect to local files, calendar, and apps. Install the filesystem server via npm:
    # Install Node.js first from nodejs.org, then:
npm install -g @modelcontextprotocol/server-filesystem
macos — Claude Desktop + MCP
  1. Download Claude for MacVisit claude.ai/download, choose the macOS build, and drag to your Applications folder.
  2. Allow it past GatekeepermacOS may flag it. Go to System Settings → Privacy & Security and click Open Anyway.
  3. Sign in and enable ToolsSettings → Features → enable Tools & Computer Use.
  4. Add filesystem MCP accessLet the agent read and write your files:
    brew install node
npm install -g @modelcontextprotocol/server-filesystem
  5. Test with a real taskTry: Read my Downloads folder and list files I haven't opened in 30 days.
linux — CLI + open-source agents
  1. Check Python 3.10+Run python3 --version. If needed: sudo apt install python3
  2. Install an agent frameworkAutoGen and CrewAI are beginner-friendly:
    pip install autogen-agentchat  # Microsoft AutoGen
# or
pip install crewai              # CrewAI
  3. Get an API keySign up at console.anthropic.com or platform.openai.com and copy your key.
  4. Run your first agentCreate agent.py and run python3 agent.py:
    from autogen import AssistantAgent, UserProxyAgent

assistant = AssistantAgent("assistant")
user      = UserProxyAgent("user", human_input_mode="NEVER")
user.initiate_chat(assistant, message="What's 2+2?")
mobile — iOS & Android
  1. Download the Claude appSearch "Claude AI" on the App Store (iOS) or Google Play (Android). Free to download.
  2. Create your accountSign up with email or Google. The free plan covers plenty of daily use.
  3. Use voice for hands-free goalsTap the mic icon and speak your task — great for "draft a reply to my last email" or "find a coffee shop nearby."
  4. Upload files and photosTap the attachment icon to share a PDF, receipt, or screenshot — it can summarize, extract data, or answer questions about it.
  5. Upgrade for heavier useClaude Pro (~$20/mo) gives priority access, longer context, and more tool usage — ideal for daily workflows.

Most reputable agents ask for explicit permission before accessing anything. They operate within a sandbox — they can only touch what you've allowed. Start with low-permission use (web search only) and gradually grant more access as you get comfortable. Never share bank passwords or government IDs.
